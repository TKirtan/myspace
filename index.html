PROJECTS


  • Mirror-Bot

    • ㅤThis is Mirror-Bot which can save files from Direct URL, YT-DL Links, Torrent Files, Magnet Links and Telegram Files to Google Drive.

    Project Details


    • Mirror-Bot downloads file from the source to server which is deployed over AWS EC2 Instance (Configuration : RHEL Machine with 30GB Storage, t2.micro), the files are then processed as per command i.e. Zipping, Unzipping etc. Further it uploads the resulted files to Google Drive by using Google Drive API and IAM Service (Service Accounts for getting over daily upload limit of Google Drive from 720GB per day to 75TB per day)
    • Code is taken from a open-source repository and deployed using Docker over VPS
    • Technologies and APIs used in the project are Containerization (Docker), Linux (OS, RHEL), AWS (Cloud Provider for VPS), GCP (For Authentication, IAM Service and Google Drive Services), Cloudflare (Workers for projecting drive files to open web, also for streaming media content), Telegram's Application Development API (User Interface), Git (Version Control System)
    • Further updates am working on are adding a CI/CD Pipeline using Jenkins, Creating a Load Balancer for the Bot on AWS and Creating an monitoring and log saver to present data in graphical form
